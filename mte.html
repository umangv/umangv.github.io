MergeThisEmail Privacy Policy

MergeThisEmail is a Google Sheets Addon for merging data in spreadsheets into emails. The full source code for MergeThisEmail is available on GitHub. MergeThisEmail is licensed under the GNU General Public License v3.0

Because all your data is processed within directly Google App Scripts, it is not shared with any external entity and is not stored on any external server.
